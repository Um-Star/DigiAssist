# DigiAssist

An offline-first digital safety and scam awareness tool designed for low-literacy users in rural communities. 

## Features
- **Phone Safety Tips**: Audio simulated tips on keeping OTPs private.
- **Loan vs Grant**: Clear definitions and differences to prevent false loan acceptance.
- **Scam Examples**: Expandable educational list of common scams.
- **Scam Risk Checker**: Interactive yes/no questionnaire to gauge risk levels.
- **AI Scam Checker**: Rule-based text analyzer to detect scam messages.
- **Report & View Scams**: LocalStorage-based community scam reporting.
- **Dark/Light Mode**: Full responsive theming for visual accessibility.

## How to Run

1. Open a terminal in this directory (`C:\Users\umarm\3mtt\DigiAssist`).
2. Install dependencies (requires Node.js):
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Requirements
Since this application uses React, TypeScript, Vite, and Tailwind CSS, it requires **Node.js** and **npm** to be installed on your computer to compile and run the code. If you do not have them installed, you can download Node.js from [nodejs.org](https://nodejs.org).
